Featured Tutorial: First Order System Response

Introduction

First order systems are fundamental building blocks in control theory. This tutorial will show you how to create and visualize a first order system's response using ControlTheoryLib.

Step 1: Import Required Modules
from controltheorylib import System, Visualizer
Step 2: Create a First Order System
# Create a system with transfer function 1/(s+1)
sys = System(transfer_function="1/(s+1)")
Step 3: Visualize the Step Response
viz = Visualizer()
viz.plot_step_response(sys)
viz.render()
Step 4: Analyze the Results

The animation will show the step response of your system. Notice the exponential approach to the steady state value.
