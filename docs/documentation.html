Installation

ControlTheoryLib can be installed via pip:

pip install controltheorylib

For development installation:

git clone https://github.com/JortStamme/Controltheorylib.git
cd controltheorylib
pip install -e .

Quick Start

Here's a basic example of using ControlTheoryLib:

from controltheorylib import System, Visualizer

# Create a simple system
sys = System(transfer_function="1/(s+1)")

# Visualize the step response
viz = Visualizer()
viz.plot_step_response(sys)
viz.render()

API Reference

System Class

Represents a control system with various representations.

Methods:
  • __init__(transfer_function=None, state_space=None) - Initialize system
  • step_response() - Compute step response
  • bode_plot() - Generate Bode plot data
  • nyquist_plot() - Generate Nyquist plot data
Visualizer Class

Creates animations of system responses using Manim.

Methods:
  • plot_step_response(system) - Animate step response
  • plot_bode(system) - Animate Bode plot
  • plot_nyquist(system) - Animate Nyquist plot
  • render() - Render the animation
